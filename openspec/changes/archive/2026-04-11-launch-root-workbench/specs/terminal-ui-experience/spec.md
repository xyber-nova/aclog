## ADDED Requirements

### Requirement: 全局训练工作台首页必须沿用统一终端工作台设计
系统 MUST 让全局训练工作台首页成为统一 terminal workbench family 的一部分，复用现有的视觉主题、阅读结构、帮助语义和导航键位，而不是为首页实现另一套独立的交互风格。

#### Scenario: 首页展示统一的工作台层级
- **WHEN** 用户打开全局训练工作台首页
- **THEN** 页面必须展示工作区上下文、主入口列表、当前选中入口的摘要或说明，以及操作提示区域
- **AND** 用户不得只能通过猜测正文理解首页下一步可以做什么

#### Scenario: 首页遵循统一导航与帮助键位
- **WHEN** 用户位于全局训练工作台首页并按下 `Enter`、`Esc`、`q`、`?`、`j` 或 `k`
- **THEN** 系统必须继续遵循统一终端工作台的确认、退出、帮助切换和移动语义
- **AND** 该行为必须与其他终端工作台保持一致

#### Scenario: 首页与其他页面共享主题语义
- **WHEN** 全局训练工作台首页展示空状态、强调信息或可恢复 sync 提示
- **THEN** 这些语义必须使用与其他终端工作台一致的文字层级与视觉语义
- **AND** 系统不得为首页单独引入冲突的颜色或提示风格

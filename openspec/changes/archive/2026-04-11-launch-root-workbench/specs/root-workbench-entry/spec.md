## Purpose

让 `aclog` 在无参数时直接进入全局训练工作台，作为现有 `sync`、`stats`、浏览和记录查看能力的统一入口。

## ADDED Requirements

### Requirement: 系统必须让 `aclog` 无参进入全局训练工作台
系统 MUST 将无参数 `aclog` 解释为“打开当前工作区的全局训练工作台”，而不是直接打印帮助；现有子命令与帮助入口 MUST 保持可用。

#### Scenario: 无参数启动首页
- **WHEN** 用户在某个工作区目录执行 `aclog`
- **THEN** 系统必须以当前目录作为工作区尝试打开全局训练工作台
- **AND** 系统不得直接把该操作解释为 `sync`、`stats` 或其他子命令

#### Scenario: 帮助入口继续可用
- **WHEN** 用户执行 `aclog --help` 或 `aclog help`
- **THEN** 系统必须继续输出完整 CLI 帮助
- **AND** 系统不得直接进入全局训练工作台

#### Scenario: 现有子命令继续直达
- **WHEN** 用户执行 `aclog sync`、`aclog stats` 或 `aclog record ...`
- **THEN** 系统必须继续按对应子命令语义执行
- **AND** 系统不得先强制经过全局训练工作台

### Requirement: 全局训练工作台必须展示本地工作区摘要与恢复入口
全局训练工作台 MUST 展示当前工作区的本地摘要，包括训练记录概况和可恢复的 sync 批次状态；这些摘要必须建立在本地 `jj` 历史与 `.aclog/sync-session.toml` 之上，而不是依赖新的可写事实源。

#### Scenario: 工作区存在未完成 sync 批次
- **WHEN** 当前工作区存在 `.aclog/sync-session.toml`
- **THEN** 全局训练工作台必须显式展示该批次可恢复
- **AND** 页面必须展示至少一个与该批次相关的恢复入口

#### Scenario: 工作区存在已记录训练历史
- **WHEN** 当前工作区已经存在可识别的 `solve(...)` 历史
- **THEN** 全局训练工作台必须展示本地训练摘要
- **AND** 摘要至少必须能让用户理解当前工作区不是空历史状态

#### Scenario: 工作区暂无训练历史
- **WHEN** 当前工作区没有任何可识别的 `solve(...)` 历史
- **THEN** 全局训练工作台必须展示明确空状态
- **AND** 页面必须继续展示下一步可进入的工作流入口

### Requirement: 全局训练工作台必须提供进入现有工作流的统一导航
全局训练工作台 MUST 提供进入现有训练工作流的统一导航，并允许用户在完成某个子工作流后回到首页，而不要求系统把所有工作流合并为一个新的共享状态机。

#### Scenario: 从首页开始新的 sync
- **WHEN** 用户在全局训练工作台中选择开始新的 sync
- **THEN** 系统必须进入现有 `sync` workflow
- **AND** 该 workflow 的提交、恢复和显式选择语义不得因首页入口而改变

#### Scenario: 从首页恢复已有 sync
- **WHEN** 用户在全局训练工作台中选择恢复 sync 批次
- **THEN** 系统必须进入现有 sync 恢复路径
- **AND** 系统不得重新定义恢复批次的判定或合并规则

#### Scenario: 从首页进入 stats 或浏览工作台
- **WHEN** 用户在全局训练工作台中选择训练统计、文件浏览或题目浏览
- **THEN** 系统必须分别进入现有 `stats`、文件视角浏览或题目视角浏览 workflow
- **AND** 这些工作流现有的筛选、provider 页签和钻取语义必须保持不变

### Requirement: 全局训练工作台必须提供记录列表的只读查看入口
全局训练工作台 MUST 提供一个记录列表查看入口，用于在不离开首页工作台家族的前提下查看当前工作区的文件记录快照；该查看入口必须与 `record list` 使用同一解释口径。

#### Scenario: 从首页打开记录列表快照
- **WHEN** 用户在全局训练工作台中选择记录列表
- **THEN** 系统必须展示当前工作区已记录文件的只读列表快照
- **AND** 该结果必须与 `aclog record list` 的记录解释口径保持一致

#### Scenario: 从记录列表快照返回首页
- **WHEN** 用户位于首页内的记录列表只读查看页并执行返回
- **THEN** 系统必须返回全局训练工作台首页
- **AND** 系统不得把该返回动作解释为退出整个 `aclog` 进程
